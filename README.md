# NM Syntax Surgeon

## Project Overview
NM Syntax Surgeon is an AI-based tool designed to detect and correct Python syntax errors automatically.
The system allows users to enter Python code and receive corrected code.

## Features
- Detects syntax errors in Python code
- Automatically fixes formatting issues
- Simple web interface using Gradio

## Technologies Used
- Python
- Transformers
- Torch
- Gradio
- Autopep8
- Black
- ReportLab
